About the Project

Stock price prediction is challenging due to the volatile and non-linear nature of financial markets. This project uses a Long Short-Term Memory (LSTM) deep learning model to predict Google stock prices based on historical market data.

Problem Statement

Traditional statistical models fail to capture long-term dependencies present in stock market data. This project addresses the limitation by using LSTM networks, which are capable of learning temporal patterns from time-series data.

How the Model Works

  • Data Collection: Historical Google stock price data is collected from Yahoo Finance, including Open, High, Low, Close, and Volume.
  • Data Preprocessing: The dataset is cleaned, normalized using MinMaxScaler, and transformed into 60-day time steps suitable for LSTM input.
  • LSTM Model Training: A deep learning LSTM model is trained to learn long-term dependencies in stock price movements.
  • Prediction: The trained model predicts future stock prices, which are later compared with actual values.

Model Workflow

Data Collection → Data Preprocessing → LSTM Training → Prediction → Visualization

Conclusion & Future Scope

This project demonstrates the effectiveness of LSTM neural networks in time-series forecasting of stock prices. While the model performs well in predicting trends, it does not consider external economic or political factors.
